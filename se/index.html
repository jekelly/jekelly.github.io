How to add these search engines:

  • Firefox: Look for a dropdown arrow in the address bar, or right-click in search bar → "Add [Engine Name]"
  • Chrome/Edge: Go to Settings → Search engines → Manage search engines → Site search
  • Manual: Click the "Install" button next to each search engine below
